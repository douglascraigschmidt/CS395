Image Crawling Reactive Microservices App (Assignment 4)

This folder contains the Android client app and the reactive microservices projects for assignment 4 and is organized into the following two projects:
  • Image Crawler Reactive Streams Client App, which is an app that is developed and run using Android Studio.

  • Reactive Microservices Image Processing Server App, which is an app that is developed and run using IntelliJ.
